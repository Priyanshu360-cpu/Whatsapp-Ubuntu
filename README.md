# Whatsapp for Ubuntu

The given application was built with **Electron JS**.

# Running procedure

```
git clone https://github.com/Priyanshu360-cpu/Whatsapp-Ubuntu

npm i

npx electron .

```

# Shortcut Keys

```
CTRL+I -- ZOOM IN
CTRL+O -- ZOOM OUT
CTRL+R -- RESET SCREEN
CTRL+F5 -- RELOAD SCREEN
CTRL +Q -- QUIT SCREEN
```
